N puzzle

The n-puzzle is a sliding puzzle that consists of a frame of numbered square tiles in random order with one tile missing. The object of the puzzle is to place the tiles in order by making sliding moves that use the empty space.

Algorithms

All presented alogorithms work with graph search. Each graph represents a new state of the field as 2D array with (i,j) combination:
(0,0) (0,1) (0,2)
(1,0) (1,1) (1,2)
(2,0) (2,1) (2,2)
A* and IDA* algorithms use heuristic function to estimate the field. Evaluation (heuristics) is necessary to speed up the search for a solution.

Heuristics

Manhattan distance

Manhattan distance is one of the most powerful heuristics. It is the sum of the moves necessary to place each cell at its "right" place.
